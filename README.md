# 🌸 For You Florist – Floral Boutique Website

For You Florist is a modern, elegant, lavender-themed floral boutique website created to showcase handcrafted bouquets, wedding floral decorations, and event flower services. The website is designed with a soft purple aesthetic to reflect elegance, creativity, and beauty — just like flowers themselves.

---

## 🌷 Project Overview

This website represents a premium flower shop that offers custom bouquets, wedding decorations, real and artificial floral arrangements, and event décor services. It is designed to provide customers with a visually pleasing experience while keeping the layout clean, professional, and user-friendly.

The design focuses on:
- Elegant lavender color palette
- Smooth navigation
- Image-based product showcase
- Responsive card layout
- Clear service descriptions
- Professional contact section

---

## ✨ Features

### 🧭 Navigation Bar
- Sticky top navigation
- Smooth scrolling to sections
- Easy access to Home, About, Menu, Services, and Contact

### 🏠 Home Section
- Shop logo display
- Brand name heading
- Elegant tagline

### 📖 About Us
- Detailed description of the florist brand
- Focus on handcrafted bouquets
- Emphasis on customer satisfaction
- Explanation of floral design philosophy

### 💐 Menu Section
- Signature bouquet collection
- Product cards with:
  - High-quality images
  - Bouquet name
  - Description
  - Pricing
- Hover animation effect

### 🌼 Services Section
- Custom bouquet creation
- Real and artificial flower arrangements
- Wedding floral decoration
- Event & party decoration
- Image-based service presentation

### 📞 Contact Section
- Store location
- Phone number
- Email address
- Google Maps link

### 🎨 Design Highlights
- Lavender theme (#E6E6FA background)
- Purple accent colors
- Soft shadows
- Rounded cards
- Clean typography (Google Fonts – Poppins)
